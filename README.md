# Finance Dashboard

## Sobre o projeto
Este projeto é um sistema simples de controle financeiro desenvolvido para praticar lógica de programação e desenvolvimento web. O usuário pode adicionar entradas e saídas de dinheiro e visualizar o saldo total atualizado automaticamente.

## Funcionalidades
- Adicionar movimentações financeiras (entrada e saída)
- Listar histórico de gastos
- Calcular total de entradas
- Calcular total de saídas
- Exibir saldo final automaticamente

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Objetivo do projeto
Este projeto foi desenvolvido com foco em prática de:
- Manipulação do DOM
- Eventos em JavaScript
- Lógica de programação
- Estruturação de páginas web
