# Finance Dashboard

## Sobre o projeto
Este projeto é um sistema simples de controle financeiro que permite ao usuário adicionar entradas e saídas de dinheiro e visualizar o saldo total atualizado automaticamente.

## Funcionalidades
- Adicionar movimentações financeiras
- Separar entradas e saídas
- Exibir histórico de lançamentos
- Calcular saldo automaticamente

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Aprendizados
Neste projeto foi praticado:
- Manipulação do DOM
- Eventos em JavaScript
- Lógica de programação
- Estruturação de interface web
